# LED点滅
##### echo 0 > /dev/myled0  → LEDを消灯
### echo 1 > /dev/myled0  → LEDを点灯
### echo 2 > /dev/myled0  → LEDが５秒点灯し１秒消灯を３回繰り返す。LEDが開始前に点灯している場合は1秒消灯してから開始。
